# IndieBook - ฟีเจอร์ทั้งหมด

## ✅ ฟีเจอร์หลัก

### 📚 การจัดการหนังสือ
- **อัพโหลดหนังสือ**: ลากและวาง PDF เพื่ออัพโหลด
- **ห้องสมุดส่วนตัว**: แสดงหนังสือทั้งหมดในรูปแบบการ์ด
- **อ่านหนังสือ**: เปิดอ่าน PDF พร้อมการซูมและเปลี่ยนหน้า
- **ลบหนังสือ**: ลบหนังสือที่ไม่ต้องการออกจากห้องสมุด

### 🔍 การค้นหา (Search)
- **ค้นหาในห้องสมุด**: ค้นหาหนังสือจากชื่อหรือชื่อไฟล์
- **ค้นหาแบบเรียลไทม์**: ผลการค้นหาปรากฏทันทีขณะพิมพ์
- **แสดงผลชัดเจน**: แจ้งเตือนเมื่อไม่พบหนังสือ

### 📊 สถิติ (Statistics)
- **จำนวนหนังสือทั้งหมด**: นับจำนวนหนังสือในห้องสมุด
- **จำนวนหน้าทั้งหมด**: รวมจำนวนหน้าของทุกเล่ม
- **หน้าเฉลี่ย**: คำนวณหน้าเฉลี่ยต่อเล่ม
- **แสดง/ซ่อนสถิติ**: สลับการแสดงผลสถิติได้

### 👨‍💼 แดชบอร์ดผู้ดูแลระบบ (Admin Dashboard)
- **ภาพรวมระบบ**: Cards แสดงสถิติหลัก
  - หนังสือทั้งหมด
  - หน้าทั้งหมด
  - จำนวนผู้ใช้
- **ตารางจัดการ**: แสดงหนังสือทั้งหมดในรูปแบบตาราง
  - ชื่อหนังสือ
  - ชื่อไฟล์
  - จำนวนหน้า
  - วันที่อัพโหลด
- **ค้นหาในแอดมิน**: ค้นหาหนังสือในตาราง
- **ลบทีละเล่ม**: ลบหนังสือแต่ละเล่มได้
- **ลบทั้งหมด**: ลบหนังสือทั้งหมดพร้อมยืนยัน 2 ครั้ง

### 🇹🇭 ภาษาไทย
- **ทุกข้อความเป็นภาษาไทย**: ทั้งหมด 100%
- **วันที่เป็นภาษาไทย**: แสดงวันที่ในรูปแบบภาษาไทย
- **ข้อความแจ้งเตือน**: Alert ทั้งหมดเป็นภาษาไทย

## 🎨 UI/UX Features

### หน้าหลัก (Home)
- Hero section พร้อมคำอธิบาย
- Feature cards 3 ช่อง
- ปุ่มนำทางไปยังหน้าต่างๆ

### ห้องสมุด (Library)
- Grid layout responsive (1-4 คอลัมน์)
- การ์ดหนังสือพร้อมข้อมูล
- ปุ่มอ่านและลบในการ์ด
- สถิติแบบ Cards (แสดง/ซ่อนได้)
- ช่องค้นหาด้านบน
- Empty state สวยงาม

### อัพโหลด (Upload)
- Drag & drop zone
- แสดงสถานะการอัพโหลด
- แจ้งเตือนเมื่อสำเร็จหรือผิดพลาด
- รองรับ PDF เท่านั้น
- ขนาดไฟล์ไม่เกิน 50MB

### ตัวอ่าน (Reader)
- พื้นหลังสีเข้ม (Dark theme)
- แถบควบคุมด้านบน sticky
- ปุ่มซูมเข้า/ออก (50%-200%)
- แสดงเปอร์เซ็นต์การซูม
- ปุ่มเปลี่ยนหน้า (ถัดไป/ก่อนหน้า)
- แสดงหมายเลขหน้าปัจจุบัน/ทั้งหมด
- Loading และ Error state

### ผู้ดูแลระบบ (Admin)
- Stats cards พร้อมไอคอนสี
- ตารางเต็มรูปแบบ
- ช่องค้นหาพร้อมไอคอน
- ปุ่มลบทั้งหมดสีแดง
- Hover effects บนแถว
- Empty state เมื่อไม่มีข้อมูล

## 🛠 เทคโนโลยี

### Frontend
- **Vite** - Build tool เร็วมาก
- **React 18** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS 4** - Styling
- **react-pdf** - PDF viewer
- **pdfjs-dist** - PDF.js library
- **lucide-react** - Icons

### Backend
- **Hono** - เบา เร็ว
- **TypeScript** - Type safety
- **Node.js File System** - จัดการไฟล์

### Storage
- **localStorage** - เก็บข้อมูลหนังสือ
- **File System** - เก็บไฟล์ PDF
